# Incoming sources

Drop raw Markdown files here when they are ready for an agent to review and convert into curated knowledge-base content.

## Rules

- Use `.md` files only.
- Prefer lowercase kebab-case filenames.
- Add optional metadata from [raw topic template](../raw-topic-template.md) when the destination is known.
- Keep raw context, source URLs, command snippets, and open questions together.
- Do not add secrets, credentials, private customer data, or sensitive infrastructure details.
