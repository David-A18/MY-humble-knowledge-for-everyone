# Processed sources

Raw Markdown files move here after an agent converts them into curated knowledge-base content.

## Rules

- Preserve the original filename when possible.
- Add a suffix such as `-2` only when a filename already exists.
- Do not treat files here as curated documentation; they are archived source material.
- Keep the curated destination linked from the ingestion handoff or changelog entry.
