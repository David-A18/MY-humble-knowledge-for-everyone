# Sources

Raw Markdown source files live here before they are transformed into curated knowledge-base articles.

Use this area for notes copied from study sessions, operational research, transcripts, exported chat notes, rough outlines, or unfinished topic material. Files in this area are not considered polished documentation until an agent ingests them into the main knowledge base.

## Folders

| Folder | Use it for |
| --- | --- |
| [incoming](incoming/README.md) | Raw `.md` files waiting for review and ingestion. |
| [processed](processed/README.md) | Raw files already converted into curated knowledge-base content. |

## How to add raw content

1. Create a `.md` file in [incoming](incoming/README.md).
2. Use lowercase kebab-case for the filename when possible.
3. Add optional metadata from [raw topic template](raw-topic-template.md) if the destination is already known.
4. Keep the raw source as complete as possible, including useful context and original links.
5. Do not include secrets, private customer data, credentials, or sensitive infrastructure details.

## Agent workflow

Agents ingesting raw sources must follow [sources agent instructions](AGENTS.md) before editing curated documentation.
